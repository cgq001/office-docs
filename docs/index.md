## 我们在做什么

我们在建设一套面向业务系统的全场景在线编辑器产品文档，覆盖文档编辑、富文本编辑、结构化内容编辑、导出与演示等能力。

当前首个产品是 `@ap666/office-word`，定位为接近在线文档体验的富文本编辑器组件。后续会继续扩展更多产品，并统一收敛到这个文档站中。

## 当前产品

### Office Word

- 产品定位：在线文档式富文本编辑器
- 技术基础：`Vue 3 + Tiptap 3`
- 核心能力：编辑、排版、导出、打印、演示、外部上传回填
- 文档入口：[开始阅读](/office-word/introduction)

## 文档结构

- 产品介绍：说明产品定位、能力边界、适用场景
- 快速开始：提供安装与最小接入示例
- API 文档：整理 Props、事件、实例方法、类型定义
- Demo 演示：给业务方展示典型接入方式和落地范式
