# Demo 演示

在线演示已经改成独立页面，不再嵌在文档正文容器里。

## 进入演示

- 演示地址：[Office Word 在线演示](/office-word/playground)
- 适合直接发给外部用户预览
- 页面会全宽展示编辑器，体验比文档内嵌方式更合适

## 演示页包含的能力

- 真实挂载 `@ap666/office-word`
- 可直接编辑文档内容
- 支持导出 HTML、PDF、图片
- 支持本地选择图片和视频进行回填演示
- 支持插入示例文件块

## 为什么单独拆页面

VitePress 的常规文档页会受到左侧边栏、右侧目录和正文宽度限制，不适合拿来做重交互的编辑器演示。独立路由更适合：

- 展示完整编辑器宽度
- 直接对外分享
- 后续继续扩展更多交互控件

## 文档页保留什么

这个页面保留为演示说明页，后续主要放：

- 演示入口
- 场景说明
- 接入代码片段
- 版本说明和已知限制
